Instructions for Mac:


You have to call from the command line, don't double click the executable.
cd to this directory first, then type:
./RetroPod

Make sure all the 5 strangely named txt files are in the same directory as the exe. These files define the levels.

Select option 5 (17 years ago when I wrote the game, option 5 only worked on super fast computers, nowadays, just select option 5!)


Once in the main game, here are the keys:

Arrow keys to move around:
UP - both fans forward.
Left / Right - one fan forward, one fan back.
Back - both fans back


f = full screen.
w = not full screen
p = pause/unpause
esc = quit.
